# vgscript-example

Example VgScript project for trying scenes, metadata, the CLI, and editor plugins.

## Contents

- `Scenes.txt` lists the scene files in play order.
- `*.vgs.meta` files are generated sidecars produced by the `vgs` CLI.

## Try it

In VS Code or JetBrains, right-click a `.vgs` file or the project folder and use
the `VgScript` menu actions.

Edit the `.vgs` files and regenerate the metadata with `vgs fix` or
`vgs fix-all`; do not hand-edit `.vgs.meta`.
